
## Installation
```
npm install -g stan-cli
```
## Getting Started

### beautify
```
stan -b <dir>
stan --beautify <dir>
```

Example:
```
stan -b build
```

### uglify
```
stan -u <dir>
stan --uglify <dir>
```

Example:
```
stan -u build
```

### imagemin
```
stan -i <dir> [type]
stan --imagemin <dir> [type]
```

Example:
```
stan -i build
```

### sass
```
stan -s <dir> [outExt]
stan --sass <dir> [outExt]
```

Example:
```
stan -s build
```
