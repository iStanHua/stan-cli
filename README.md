
## Installation
```
npm install -g stan-cli
```
## Getting Started

### beautify
```
stan -b <path>
stan --beautify <path>
```

Example:
```
stan -b build
```

### uglify
```
stan -u <path>
stan --uglify <path>
```

Example:
```
stan -u build
```

### imagemin
```
stan -i <path>
stan --imagemin <path>
```

Example:
```
stan -i build
```
